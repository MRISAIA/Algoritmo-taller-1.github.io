Tema: HTML

¿QUÉ ES EL HTML?

HTML es un lenguaje de marcado que utiliza una serie de códigos llamados etiquetas que van definiendo los elementos que componen una página web: texto, imágenes, etc. Estas etiquetas serán interpretadas

¿PARA QUÉ SIRVE HTML?

HTML sirve como al base de cualquier sitio web. Proporciona una estructura muy simple en la que se pueden añadir elementos como estilos o funciones interactivas como usando CSS o JavaScript. Si no existiera HTML, los navegadores no tendrían la capacidad de poder ejecutar contenidos webs de manera eficiente.

¿CÓMO FUNCIONA?

HTML funciona a través del uso de etiquetas, estas etiquetas les permiten a los navegadores renderizar el contenido de una manera mas estructurada, unos ejemplos de etiquetas pueden ser:H1,H2,H3 (hasta 6) Párrafos que son encabezamientos (con distintos niveles).

USOS ACTUALES

En la actualidad, HTML sigue estando muy presente en el desarrollo web. Ya que permite crear interfaces de usuario interactivas y muy dinámicas Frameworks como Angular, React y Vue. js lo utilizan junto con JavaScript para crear aplicaciones web modernas.

TAGS QUE DEFINEN LA ESTRUCTURA DE UNA PÁGINA WEB

*h1 a h6: Representan los encabezados, donde h1 es el de mayor importancia y h6 el de menor.

*p: Define un bloque de texto o párrafo.

*pre: Muestra texto preformateado, conservando los espacios y saltos de línea.

*br: Añade un salto de línea.

*hr: Inserta una línea horizontal que separa el contenido.

*a href="": Crea un enlace hacia otra página o recurso.

*img src="" width="" height="": Inserta una imagen en la página con opciones de tamaño.

*style: Permite aplicar directamente estilos CSS, como colores o tamaños de fuente.

*sub: Presenta texto como subíndice.

*sup: Muestra texto como superíndice.

*small: Hace que el texto aparezca en un tamaño más pequeño.

*i: Representa el texto en cursiva.

*table: Define una tabla con filas y columnas.

*ul, ol, li: Crean listas, ya sea ordenadas (ol) o desordenadas (ul), con elementos (li).
